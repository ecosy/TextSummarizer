# TextSummarizer
인턴 개인과제를 위해 만든 텍스트 요약 알고리즘입니다.<br>Textrank와 PMI 기법을 활용한 추출요약 방식을 사용했습니다.<br>Textrank, PMI에 관한 설명, 오픈소스코드는 [해당 링크](https://bab2min.tistory.com/552)를 참조하시면 됩니다.<br>
해당링크의 오픈소스코드를 변형하고 새로운 알고리즘을 추가하여 만들었습니다.
